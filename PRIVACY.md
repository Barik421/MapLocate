# MapLocate Privacy Policy

Last updated: July 22, 2026

MapLocate helps users find selected settlements, villages, cities, and addresses on an interactive map.

## Data MapLocate Processes

MapLocate processes:

- selected text that the user asks MapLocate to find;
- search text typed in the MapLocate side panel;
- extension settings such as language, theme, and selection behavior;
- recent locations saved locally by the extension for faster repeated searches.

## How Data Is Used

Selected text and search queries are used only to provide MapLocate's location lookup features.

MapLocate sends location search queries to Nominatim/OpenStreetMap to retrieve place results and map information. Map tiles are loaded from OpenStreetMap tile servers. When the user clicks Open in Google Maps or Open in Google, MapLocate opens Google services with the selected place query.

## Storage

Settings are stored using `chrome.storage.sync`, so Chrome may sync them across the user's signed-in browsers. Recent locations are stored locally using `chrome.storage.local`.

## Analytics And Advertising

MapLocate does not include advertising, does not serve personalized ads, and does not run its own analytics or tracking service.

## Data Sharing

MapLocate does not sell user data. Search queries are shared only with the external services needed to provide the user-requested map and search features:

- Nominatim/OpenStreetMap for place lookup;
- OpenStreetMap tile servers for map display;
- Google Maps or Google Search only when the user clicks those actions.

Those services process data according to their own privacy policies.
